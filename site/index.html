SynthEddy Documentation

This is the documentation for the SynthEddy, a tool to generate 3D velocity fields consist of synthetic eddies, mimicking turbulent flows, and to be used as initial conditions by turbulent CFD simulations. The documentation follows a rational document driven process (Parnas and Clements, 1986), as presented for research software in the second chapter of Software Engineering for Science, and covered in the graduate class Development of Scientific Computing Software.

The software requirements specification is based on a template presented in SmithEtAl (2007) and Smith And Lai (2006). The design documentation (Parnas-inspired module guide and module interface specification) is based on a template presented in ElSheikh et al (2004).
